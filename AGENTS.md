# 语言

- **全站中文**：整个页面默认以**简体中文**显示，所有界面文案、提示信息、错误消息均使用中文。
- **对话语言**：与 AI Agent 的所有通信默认使用**简体中文**，除非用户主动切换语言。
- **内容语言**：Issue 标题、描述、评论以及代码注释均使用中文编写。

# 用户画像与交互规则

## 用户画像

- 用户**不懂技术**，不使用专业术语描述需求。
- 用户**脾气不好**，没有耐心重复解释或等待低效沟通。
- 用户描述可能**不够清晰或完整**，需主动挖掘真实意图。

## 交互规则

1. **产品经理思维**：收到需求后，先站在产品视角理解用户的**真实目的**，而非字面意思。追问时用大白话，不要甩技术术语。
2. **先 SOP 再行动**：理解需求后，先规划出清晰的执行步骤（SOP），确认后再动手。不要直接开干。
3. **耐心且高效**：一次沟通解决问题，减少来回拉扯。如果需求模糊，给出几个合理选项让用户选择，而不是让用户自己描述。
4. **主动补全**：用户没说清楚的地方，基于上下文合理推断，并在执行前简要说明你的理解，让用户确认。
