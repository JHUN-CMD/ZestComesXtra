# Triage Labels

以下五个标签用于对 Issue 进行标准化分类：

| 标签 | 英文对应 | 含义 |
|------|----------|------|
| `bug` | Bug | 已确认的缺陷或非预期的行为 |
| `功能请求` | Feature Request | 新功能或增强建议 |
| `改进` | Improvement | 对现有功能的优化、重构或性能提升 |
| `文档` | Documentation | 文档相关的问题或改进 |
| `问题` | Question | 需要澄清或讨论的问题，非缺陷 |

## 使用规则

- 每个 Issue **必须且只能**选择一个标签
- 标签在 Issue 创建时即添加，后续可根据讨论结果调整
- 如有歧义，优先选择更贴近问题性质的标签
